  \section Pong Game made for GAME3121 at George Brown College F2021
   
   This game demonstrates the use of object-oriented programming in C++ programming language through the development of the game Pong. It was created for the Game Engine Development class at George Brown College GAME3121 for the fall 2021 semester. 
 
  @author Eric Galway

  @note This game is made with Ogre Game Engine
  @attention The controls for the game are WASD for movement and R -key to restart the game
