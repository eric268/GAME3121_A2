Doodle jump game
Main Page

Game made for GAME3121 A2 at George Brown College F2021

This game demonstrates the use of object-oriented programming in C++ programming language through the development
of the doodle platformer game. It was created for the Game Engine Development class at George Brown College GAME3121 for the Fall 2021
semester.

The objective of the game is reach the highest platform with without losing all 3 of your starting lives. If all lives are lost
before reaching the top the player is able to restart the game.

Author
Eric Galway
Note
This game is made with Ogre Game Engine
Attention
The controls for the game are:
- A Move left
- D Move right
- R Restart Game
- ESC To Quit
