◆ GetGravityScale()

float PhysicsBody::GetGravityScale ( )

Returns the gravity scale multipler

This variable acts as a multiplier to the current gravity of 9.8 which affects physics bodies.
A higher gravity scale increases gravity while a lower gravity scale reduces the effects of gravity

Returns
a floating point variable
